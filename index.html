Version History

v3.1.0 - April 02, 2026

Search & Productivity Update
  • **New Features**:
    • Added Search field to jump directly to specific ARBIT or EHR numbers within loaded data.
    • Added a "Help" button with a comprehensive mapping guidelines table for NSA and STATE statuses.
    • Separated "Dates Output" and "Claim Output" in Arbit Case Notes for independent copying.
    • Added UB04 under file name generator section.
  • **UI/UX Improvements**:
    • Implemented a highlight glow effect on output fields when a search is successful.

v3.0.0 - March 31, 2026

Major Update & UI/UX Overhaul
  • **New Features**:
    • Added NUTEX (GoRev Portal) support for missing case notes, including a dedicated "Multiple CPT Code" option.
    • Introduced a new "Itemized EOB Notes" section specifically for the NUTEX portal, featuring a three-option radio group for detailed EOB responses.
    • Implemented new color blind support themes:
      • **Trichromacy**: High-contrast theme for enhanced readability.
      • **Dichromacy**: Blue-yellow/orange focused theme for common red-green color blindness.
      • **Monochromacy**: Grayscale theme for total color blindness.
  • **UI/UX Improvements**:
    • Added "press" animation and visual feedback (color change & "Copied!" text) to all copy buttons.
    • Converted NUTEX missing notes to radio buttons for mutual exclusivity and consolidated output into a single field.
