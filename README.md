# Document Text Generator

A simple web tool that generates standardized document filenames using a **Dispute Number** and **EHR Number** to help speed up document processing workflows.

---

## 🚀 Live Demo

Access the tool here:

https://mrplayground.github.io/ArbitFileNameGenerator/


---

## ✨ Features

- Instant filename generation
- One-click copy buttons
- Clear button to reset fields
- Real-time text updates
- Simple and lightweight design
- No installation required
- Runs directly in browser

---

## 📋 How To Use

1. Enter the **Dispute Number**
2. Enter the **EHR Number**
3. Copy the generated filename needed
4. Click **Clear** to reset inputs

---

## 📄 Generated Document Formats

| Document | Generated Format |
|----------|------------------|
| HCFA | dispute_ehr_hcfa |
| EOB | dispute_ehr_physicaleob |
| HMO | dispute_ehr_insurancecard |
| OP | dispute_ehr_opreport |
| OP2 | dispute_ehr_opreport2 |
| OP3 | dispute_ehr_opreport3 |

---

## 📷 Example

Input:
```
Dispute Number: 123123
EHR Number: 321321
```

Output:
```
123123_321321_hcfa
123123_321321_physicaleob
123123_321321_insurancecard
123123_321321_opreport
123123_321321_opreport2
123123_321321_opreport3
```
